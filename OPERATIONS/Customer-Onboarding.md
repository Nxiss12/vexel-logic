## Day 2-3: Kickoff Call & Setup

### Before the Call

**Prepare:**
- [ ] Review their business (website, Google reviews)
- [ ] Note industry (trades, healthcare, etc.)
- [ ] Check their plan tier
- [ ] Prepare demo environment

### During Kickoff Call (45 min)

**Minutes 0-10: Discovery**
- "Tell me about your biggest admin headache right now"
- "How many customer inquiries per week?"
- "What tools are you currently using?"

**Minutes 10-30: Configuration**

**Tool 1: Missed Call Bot**
1. Add their business phone number
2. Configure SMS templates
3. Test with live call
4. Show them the dashboard

**Tool 2: Review Engine**
1. Connect Google Business Profile
2. Set review request timing (7 days after job)
3. Customize email template
4. Show example

**Tool 3: Unified Inbox**
1. Connect email, SMS, WhatsApp
2. Show inbox interface
3. Explain notification system

**Minutes 30-40: Training**
- Walk through each tool
- Answer questions
- Show where to get help

**Minutes 40-45: Next Steps**
- Set Day 7 check-in call
- Share video tutorials (if available)
- Confirm contact preferences

## Day 7: First Check-In

### Check-In Call (15 min)

**Questions to ask:**
1. "Have you seen any results yet?"
2. "Is everything working as expected?"
3. "Any features you're not using? Why?"
4. "What else would make this more valuable?"

**Show them:**
- Usage statistics
- Any leads captured
- Any reviews received

**Goal:** Identify and fix any blockers

## Red Flags (At-Risk Customers)

**Watch for:**
- No login after 7 days
- Zero tool usage after 14 days
- Missed onboarding calls
- Multiple support tickets early
- Negative feedback in calls

**Action if red flag:**
1. Call immediately (don't email)
2. Identify specific problem
3. Offer hands-on help
4. Consider discount/credit if needed
5. Set clear success criteria

---

## Success Metrics

**Track for each customer:**
- Time to first value (target: <7 days)
- Tool adoption rate (target: >80%)
- Satisfaction score (target: 8+/10)
- Support tickets (target: <2 in first 30 days)
