- Aims and Objectives
- Aim:
- introduce basic principles of programming and algorithms
- Objectives:
- teach fundamental programming constructs , types and
variables, expressions, control structures and functions in C
- teach how to design and analyse simple algorithms and data
structures , allowing efficient storage and manipulation of data
- familiarise students with software development methods ,
including documentation, testing, debugging, and the use of
software tools, such as version control systems and continuous
integration
