## ‚ö° Quick Start

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/VexelLogic.git
cd VexelLogic
```

### 2. Install Backend

```bash
cd backend/
npm install
cp ENV_SETUP.md .env
# Edit .env with your API keys
npm start
```

### 3. Deploy Frontend

```bash
# Option A: Vercel (recommended)
cd landing-page/
vercel

# Option B: Your own server
# Upload landing-page/ to web server
```

### 4. Configure Stripe

1. Create products in [Stripe Dashboard](https://dashboard.stripe.com)
2. Add Price IDs to `.env`
3. Set up webhook endpoint

### 5. Test & Launch!

Full instructions: [`DEPLOYMENT_GUIDE.md`](DEPLOYMENT_GUIDE.md)
